## 💻 Terminal Commands

Available commands in the terminal window:
- `help` — list commands
- `about` — personal info
- `skills` — tech stack
- `projects` — project list
- `contact` — contact info
- `whoami` — identity
- `easter` — secret
- `clear` — clear screen
